Manual Steps Required

These require human attention — the install scripts cannot automate them.
  1. Tokens/secrets: Ensure ~/.smartclaw/openclaw.json is configured before running install. bootstrap.sh handles this for new machines.
  2. Qdrant Docker: ai.smartclaw.qdrant requires Docker running. Run docker info before installing. Service starts on next login if Docker is not currently running.
  3. AO dashboard directory: Requires $HOME/projects_reference/agent-orchestrator/packages/web. Install the AO repo first, or set AO_DASHBOARD_DIR before install.
  4. Old hardcoded plists: If you have pre-migration plist files in ~/Library/LaunchAgents/, unload them first: launchctl bootout gui/$(id -u)/<label>
  5. Migrated gateway cron jobs: After migration, the jobs in ~/.smartclaw/cron/jobs.json are set to enabled: false. This is correct — do not delete them (the disabled state is the migration marker).
